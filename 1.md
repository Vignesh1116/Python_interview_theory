1. What is Python?

Python is a high-level, interpreted, general-purpose programming language designed to emphasize code readability and simplicity. It supports multiple programming paradigms such as procedural programming, object-oriented programming, and functional programming.

Python was created by Guido van Rossum and first released in 1991.

Key characteristics:

Simple syntax similar to English

Dynamically typed language

Automatic memory management

Large standard library

Cross-platform support

Python is widely used in:

Web development

Data science

Artificial intelligence

Automation

Cloud computing
