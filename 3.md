3. What is the difference between Python 2 and Python 3?

Python 3 is the improved and modern version of Python, while Python 2 is now deprecated.

Major differences include:

Feature	Python 2	Python 3
Print Statement	print "Hello"	print("Hello")
Division	Integer division	Float division
Unicode	Limited support	Full Unicode support
Support	No longer maintained	Actively maintained

Python 3 is recommended for all modern applications.
