2. What are the main features of Python?

Python provides several powerful features that make it popular in the software industry.

Major features include:

1. Easy to Learn
Python syntax is simple and readable.

2. Interpreted Language
Python code is executed line by line using an interpreter.

3. Dynamically Typed
Variable types are determined at runtime.

4. Object-Oriented
Python supports classes, objects, inheritance, and polymorphism.

5. Large Standard Library
Python provides built-in modules for many tasks.

6. Portable
Python programs run on multiple operating systems without modification.

7. Open Source
Python is freely available and supported by a large developer community.
